QR Code Types Explained

  • Website URL: Links directly to a website when scanned
  • Plain Text: Displays text message when scanned
  • Email: Opens email client with pre-filled address, subject, and body
  • Phone: Initiates a call to the specified number when scanned
  • SMS: Opens message app with pre-filled number and text
  • vCard: Adds contact information to the scanner's address book
  • WiFi: Automatically connects to a WiFi network
  • Geo Location: Opens map application to specified coordinates
