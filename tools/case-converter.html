About Text Cases
  • UPPERCASE

    All characters are converted to uppercase letters.

  • lowercase

    All characters are converted to lowercase letters.

  • Title Case

    The first letter of each word is capitalized.

  • Sentence case

    The first letter of each sentence is capitalized.

  • Capitalized Case

    Only the first letter of the entire text is capitalized.

  • camelCase

    First word lowercase, then first letter of each word capitalized with no spaces.

  • PascalCase

    First letter of each word capitalized with no spaces.

  • snake_case

    All lowercase with underscores between words.

Use Cases
  • Programming - Convert variable names between different naming conventions (camelCase, snake_case, etc.)
  • Document Editing - Format titles, headings, or text for consistency
  • Marketing - Create proper title case for headlines and email subjects
  • Data Cleaning - Standardize text data for database entries
